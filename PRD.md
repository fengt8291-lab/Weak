# PRD - 测测你的人性弱点 (Weak)

> 本文档定义「测测你的人性弱点」产品的完整产品需求，基于卡内基《人性的弱点》设计。

---

## 一、产品概述

### 1.1 产品定位

**一句话描述：** 一款基于卡内基《人性的弱点》的AI性格测试工具，通过25道选择题诊断你在人际交往中的"致命弱点"。

### 1.2 核心价值主张

> "25道题，测出你在人际交往中的致命短板"

### 1.3 灵感来源：卡内基六大弱点

| 维度 | 卡内基原文核心 | 说明 |
|------|--------------|------|
| **批评型** | "不要批评、指责或抱怨" | 习惯性挑剔，总看到别人缺点 |
| **冷漠型** | "真诚地关注他人" | 只在乎自己，忽视对方感受 |
| **自恋型** | "谈论对方感兴趣的话题" | 聊天只说自己，不问对方 |
| **记不住型** | "记住对方的名字" | 经常忘记别人名字和信息 |
| **防御型** | "在争论中获胜的唯一方法是避免争论" | 遇到分歧必须赢 |
| **索取型** | "激发他人内在需求" | 只关注自己想从对方那里得到什么 |

---

## 二、已实现功能

### 2.1 测试系统

- **25道题目**：混合程度题(5选项) + 常规题(5选项)
- **场景分布**：职场12题 + 校园6题 + 生活7题
- **积分机制**：每题 -1~+2分，每维度3-5题
- **健康指数**：0-100分，反映整体人际健康程度

### 2.2 结果页

- **六维雷达图**：SVG实现，6维度可视化
- **详细分析卡片**：150+字深度分析 + 卡内基原文引用
- **卡内基原典**：每个维度都有书中原文引用
- **行动建议**：3条具体可执行改变方法
- **安全区展示**：得分≤2的维度标记为健康
