# 题库设计 - 测测你的人性弱点 (Weak)

> 25道场景化选择题，测试六大维度：批评型、冷漠型、自恋型、记不住型、防御型、索取型

---

## 一、积分机制

### 1.1 计分规则

每道题4个选项，每个选项对应一个维度，分值范围 **-1 ~ +2**：

| 分值 | 含义 |
|------|------|
| -1 | 最健康的选择（完全避开该弱点） |
| 0 | 中性/边缘选择 |
| +1 | 轻度表现该弱点 |
| +2 | 重度表现该弱点 |

### 1.2 维度计分公式

```
维度得分 = Σ(该维度题目分值)
```

25道题，6个维度分配：
- 批评型：4题（Q1, Q5, Q10, Q16）
- 冷漠型：4题（Q2, Q8, Q13, Q19）
- 自恋型：4题（Q3, Q9, Q14, Q20）
- 记不住型：4题（Q4, Q11, Q17, Q23）
- 防御型：4题（Q6, Q12, Q18, Q24）
- 索取型：4题（Q7, Q15, Q21, Q25）
- **平衡题：1题（Q22）** — 不计入任何维度，只作为干扰项

### 1.3 结果判定

- **主要弱点**：得分最高的维度（可能并列）
- **次要弱点**：得分第二高的维度
- **安全区**：得分 ≤ 0 的维度视为健康
- **危险区**：得分 ≥ 6 的维度视为严重

## 三、维度积分速查表

| 维度 | Q编号 | 满分 | 说明 |
|------|-------|------|------|
| 批评型 | Q1, Q5, Q10, Q16 | 8分 | 越高说明越爱批评挑剔 |
| 冷漠型 | Q2, Q8, Q13, Q19 | 8分 | 越高说明越冷漠疏离 |
| 自恋型 | Q3, Q9, Q14, Q20 | 8分 | 越高说明越自我中心 |
| 记不住型 | Q4, Q11, Q17, Q23 | 8分 | 越高说明越忽视他人 |

- 0-2分：健康范围，该维度表现良好
- 3-5分：轻度信号，需要注意
- 6-8分：**核心弱点**，该维度是主要人际障碍

## 五、题目设计原则

1. **场景真实**：每道题都是日常人际场景，避免抽象
2. **选项平等**：每个选项看起来都合理，不暴露正确答案
3. **维度互斥**：每道题只测一个维度，避免混淆
4. **文化适配**：场景符合中国职场/校园/家庭环境
5. **平衡设计**：每个维度有4题，积分权重相等
6. **防作弊**：不能通过"全选温柔选项"来规避弱点检测
