# 你的人际关系隐形裂隙 (Gap)

基于卡耐基《人性的弱点》的人际弱点测试网站。

## 产品概述

通过25道场景化测试题（Q1-Q19选择题 + Q20-Q25七级量表），诊断你在人际交往中的6大弱点维度：

- 🔍 批评型 - 习惯性挑剔，总看到别人缺点
- ❄️ 冷漠型 - 只在乎自己，忽视对方感受
- 🗣️ 自恋型 - 聊天只说自己，不问对方
- 🧠 记不住型 - 经常忘记别人名字和信息
- 🛡️ 防御型 - 遇到分歧必须赢，无法接受不同意见
- 🤲 索取型 - 只关注自己想从对方那里得到什么

## 积分规则

- **选择题（Q1-Q19）**：A=-1, B=0, C=0, D=+1, E=+2，最低为0分
- **量表题（Q20-Q25）**：1-7选项对应0-6分，线性计分
- **总分上限**：70分（六大维度各有名自上限）
- **健康指数**：100 - (总分/70×100）

详细规则见 [QUESTION_BANK.md](QUESTION_BANK.md)

## 技术栈

- Flask (Python)
- SQLite 数据库
- 原生 HTML/CSS/JS（无框架依赖）
- 前端独立计分，无需后端计算

## 快速开始

```bash
cd /var/www/gap
pip install flask
python app.py
# 访问 http://localhost:5002
```

## 访问地址

- 本地：http://localhost:5002
- 演示模式：任意通行码均可进入

## 部署

```bash
# 生产环境
pip install gunicorn
gunicorn -w 2 -b 0.0.0.0:5002 app:app
```
